군집화(Clustering)  : 어떤 대상들을 구분해서 그룹을 만드는 것
* 분류(Classification) : 어떤 대상이 어떤 그룹에 속하는지를 판단하는 것

1000만개의 관측치(행)를 입력하고 100개의 클러스터가 필요하다고 알려주면, 유사한 속성을 가진 관측치끼리 분류하여
총 100개의 클러스터를 만들어준다. 

군집화는 서로 가까운 관측치를 찾아주는 머신러닝의 기법.

좌표상에서 가깝다는 것은 데이터가 서로 비슷하다는 의미와 같다.

연관규칙학습(Association rule learning) : 서로 연관된 특징을 찾아내는 것(장바구니 분석)

ex) 라면을 구입한 사람은 계란을 구입할 확률이 높다. 서로 연관성(Association)이 높다.
    '추천'이라는 단어가 뒤에 붙은 것들은 거의 다 연관규칙을 이용한 것이라고 보면 된다.


비슷한 관측치(행)을 그룹핑 해주는 것 -> 군집화
서로 관련이 있는 특성(열)을 그룹핑 해주는 것 -> 연관규칙

비지도학습은 탐험적이다. 데이터들의 성격을 파악하는 것이 목적.
독립변수와 종속변수의 구분이 중요치 않다. 데이터만 있으면 된다.

지도학습은 역사적이다. 과거의 원인과 결과를 바탕으로 결과를 모르는 원인이 발생했을 때 결과를 추측하는 것이 목적.
독립변수(원인)과 종속변수(결과)가 필요하다.
